# Nuxt Flags E2E Tests

End-to-end tests for the Nuxt integration using Playwright with [`@nuxt/test-utils`](https://nuxt.com/docs/getting-started/testing).

## Running Tests

```bash
# Run all tests
pnpm test:e2e

# Run tests in UI mode
pnpm test:e2e:ui
```

## Configuration

The test suite uses `@nuxt/test-utils/playwright` for Nuxt-specific testing utilities. (See [docs](https://nuxt.com/docs/4.x/getting-started/testing#testing-in-a-browser).)
